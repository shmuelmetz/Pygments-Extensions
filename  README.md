# pygments-extensions

`pygments-extensions` is a collection of additional lexers and related helpers
for [Pygments](https://pygments.org/), focused on legacy and niche languages
such as:

- IBM PL/I
- High Level Assembler (HLASM)
- Classic and modern REXX dialects
- NetRexx
- ooRexx
- Regina REXX

The goal is to provide pragmatic, reasonably accurate highlighting for code
found in mainframe, midrange, and migration contexts.

## Installation

```bash
pip install pygments-extensions
